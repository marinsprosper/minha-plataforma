## Rodar local
1) Instale Node.js LTS
2) Crie um arquivo `.env` com:
- JWT_SECRET
- EULEN_API_TOKEN (quando tiver)
- PLATFORM_DEPIX_ADDRESS
- ADMIN_EMAIL

3) Rode:
```bash
npm install
npm run dev
```

Abra:

- /register.html
- /login.html
- /app.html
- /admin.html
